# Prof
C'est un projet qui fournira de l'aide aux profs de gestion de l'enseignement secondaire.
vous allez trouvez des fiches de déroulement des séances de 2éme année secondaire et d'autres documents que j'ai jugé utiles pour l'enseignement.Soiyez les bienvenues pour ajouter tout type de document qui peut aider les collègues à améliorer leurs experiences d'enseignement.

vous pouvez copier ces fiches sur votre ordinateur et les ameliorer puis les partager pour contribuer à maintenir à jour cette base de données.

merci.
